# A Simple Note Apps

A simple note apps

## Run Locally

Clone the project

```bash
  git clone https://github.com/rizrmdhn/note-app.git
```

Go to the project directory

```bash
  cd note-app
```

Install server dependencies

```bash
  npm run server:init
```

Install client dependencies

```bash
  npm run client:init
```

Start the server in dev mode

```bash
  npm run server:dev
```

Start the client in dev mode

```bash
  npm run client:dev
```
